Scoring Engine

Five Signals. One Score.

Every dimension a senior headhunter evaluates — quantified, weighted, and fused into a single auditable score.

30% weight

Semantic Fit

Measures meaning-level alignment between the job description and candidate resume using vector embeddings. Captures intent and context, not just keyword matches.

25% weight

Skills Match

Evaluates depth and adjacency of technical skill overlap. Vue.js expertise scores against a React requirement because our adjacency map understands skill transferability.

20% weight

Career Trajectory

Piecewise scoring that evaluates promotion rate, seniority achieved, and title progression — without penalizing long tenures or new graduates.

15% weight

Behavioral Signals

Profile completeness, engagement freshness, response speed, and portfolio depth — all mapped to explicit, auditable data fields.

10% weight

Domain Alignment

Measures whether a candidate's vocabulary, projects, and tech stack genuinely match the hiring domain — purely content-based, zero bias.
